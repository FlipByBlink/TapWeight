#  TapWeight 体重メモ

Tool for fastest recording body weight to Apple Health app.

【OPTION】

With body fat percentage.

Automatically launch Apple Health app after record.

Unit: kg, lbs, st


==== Japanese(native) ====

Appleヘルスケアアプリに体重を最速で登録するためのアプリです。

【オプション】

体脂肪率も一緒に登録。

データ登録後に自動的にAppleヘルスケアアプリを立ち上げ。

単位: kg, lbs, st
